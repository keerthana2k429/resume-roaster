🔥 Resume Roaster

Resume Roaster is a modern web application that analyzes resumes in two unique modes: Roast Mode and Optimizer Mode.
Built using HTML, CSS, and JavaScript, it provides instant resume feedback directly in the browser without requiring a backend server or API key.

🚀 Features

🔥 Roast Mode
- Provides humorous and constructive resume critiques
- Detects weak phrases and generic wording
- Identifies missing achievements and measurable results
- Highlights common resume mistakes

✨ Optimizer Mode

- Generates ATS-friendly suggestions
- Improves resume bullet points
- Recommends stronger action verbs
- Provides professional wording enhancements

📊 Resume Analytics

- Word count
- Character count
- Resume statistics
- Content quality feedback

🎨 Modern User Interface

- Dark-themed responsive design
- Smooth animations and transitions
- Typewriter-style output effect
- Copy-to-clipboard functionality

⚡ Fully Client-Side

- No API key required
- No backend server required
- Runs completely offline
